# HDG Bootcamp App!

Bu uygulama, Huawei Developer Groups tarafından düzenlenen eğitimlerin listesini gösterir.


# Kullanılan Teknolojiler

 - Kotlin 
 - API com Retrofit 
 - Jetpack Compose 
 - Clean Architecture 
 - MVVM
 - HMS
 - Huawei Auth Services
